# V3.1 Frontend Review

## Review Position

V3.1 is the current main learner version. It keeps the V3 information architecture and military terminal direction while removing weak generated course imagery. The product should feel like a serious mobile training terminal: direct next action, readable course material, clear question practice, and explainable ability feedback.

## User Stories Used For Review

- As a learner, I can register/login and immediately know what to continue today.
- As a learner, I can enter a week, understand its goal, open a lesson, read the manual/deep dive, and mark it done.
- As a learner, I can practice questions by course or skill domain and leave real records.
- As a learner, I can understand that the Profile radar comes from real attempts, not mocked data.
- As an admin, I can inspect users, progress, and question performance without editing data.

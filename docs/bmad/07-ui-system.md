# UI System

## Direction

Military, cold, hard, and technical. The interface should feel like a training terminal, not a marketing site.

## Tokens

- Base: black steel and graphite.
- Surface: gunmetal panels.
- Accent: controlled warning yellow.
- Success: low-saturation tactical green.
- Error: restrained dark red.

## Components

- Buttons use hard 4px corners and clear borders.
- Panels use thin lines, low contrast metal surfaces, and status labels.
- Data modules use compact labels, numeric readouts, and clear source text.
- Radar uses restrained SVG lines and real values only.

## Prohibited

No purple-blue gradients, no cheap neon, no soft card stacks, no decorative fake analytics.

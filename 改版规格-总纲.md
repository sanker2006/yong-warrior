# 改版规格书 · 第一部分：总纲（V2 重写版）

> **重要**：本规格书基于 `00-训练总纲.docx` 及6份教案文档的完整内容编写。
> 
> **适用对象**：激光对抗/战术游戏团队的军事化训练体系（非保安培训）

---

## 一、项目定位

### 1.1 这是什么

一套 **30周军事战术训练课程** 的数字化学习平台（Web App）。

**受训对象**：激光对抗游戏团队成员（老兵骨干30人 + 新入队成员）
**训练形式**：每周1次集中训练（周末，游戏后2-3小时）
**核心内容**：射击基础 → 单兵战术 → 班组配合 → SOP作战流程 → 综合对抗 → 精英打磨
**参考来源**：美军FM 3-21.8、USMC MCWP 3-35.3、解放军步兵战术教材、三三制战术研究

### 1.2 核心教学理念

| 理念 | 含义 |
|------|------|
| **渐进式技能建设** | 6个阶段递进，从静态到动态、从个人到班组、从练习到实战 |
| **OODA循环贯穿** | 观察→判断→决策→行动，每个阶段都强化这个循环 |
| **双轨战术体系** | 班级层面用美军火力组制（Bounding Overwatch），小组层面用解放军三三制 |
| **SOP标准化** | 进攻/防御/建筑清搜/特殊任务各有标准操作流程 |
| **AAR复盘文化** | 每次对抗后执行标准复盘：事实→问题→改进 |

## 五、验收标准

### 5.1 内容准确性验收（最关键！）

- [ ] **每课 lessonManuals 的所有技术术语、数据标准、时间要求必须与教案完全一致**
  - 例如：W2 的"三姿切换时间标准"必须是 立↔跪2秒、跪↔卧2秒、立↔卧3秒
  - 例如：W8 的"3秒冲刺暴露时间"必须是 ≤3秒
  - 例如：W11 的"Bounding Overwatch跃进距离"必须是 8-15米
  - 例如：W12 的"三三制跃进距离"必须是 8-12米
  - 例如：W13 的"SALUTE报告时限"必须是 30秒内
  - 例如：W15 的"进攻SOP总时间标准"必须是 ≤4.5分钟
  - 例如：W17 的"建筑清搜标准"必须是 5人小组3室≤3分钟零误伤
  - 例如：W24 的"战损比标准"必须是 ≥1:2

- [ ] **所有战术概念的解释必须有实操层面的深度**
  - ❌ 错误："三三制是一种三人战斗编组方法"（太浅，学员看完不知道怎么用）
  - ✅ 正确：解释三三制的四个原理（三点成面/360°覆盖/最小指挥单元/战损容忍）+ 三种队形（前三角/后三角/一字展开）+ 动态角色轮换方法 + 与Bounding Overwatch的区别 + 适用和不适用场景 + 常见错误

- [ ] **所有情景判断题的场景必须是激光对抗/战术游戏的实际场景**
  - ✅ 正确场景：CQB中Fatal Funnel处置、Bounding Overwatch时掩护组被压制怎么办、三三制遭遇战的3秒反应、进攻SOP中突破时机选择
  - ❌ 错误场景：门岗访客登记、消防通道违停、电梯困人（这些跟本课程完全无关）

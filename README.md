# 甬士训练 H5

甬士训练是面向课程学习、题库训练、活动报名和后台态势查看的 H5 训练系统。当前稳定入口仍为 V3.2，V5 为移动端与管理端重建版本，已接入 V3.2 题库数据契约并修复活动报名身份隔离问题。

旧版 `v2.html`、`v3.html`、`v3.1.html`、`v4.html` 保留为历史参考，不作为默认演示入口。

## 本地启动

```bash
npm run start
```

启动后访问：

- 学员端：`http://127.0.0.1:4317/v3.2.html`
- 后台端：`http://127.0.0.1:4317/admin.html`
- V5 学员端：`http://127.0.0.1:4317/v5.html`
- V5 管理端：`http://127.0.0.1:4317/admin-v5.html`
- 根路径 `/` 会自动跳转到 V3.2 学员端。

不要用 `file://` 验收注册、登录、题库提交、Profile、后台等 API 行为。

## 数据与运行时

- 主存储：SQLite，文件位于 `data/app.db`。
- 旧 JSON 文件仅用于首次迁移和备份参考，运行时写入 SQLite。
- Node 使用内置 `node:sqlite`，启动或测试时出现 experimental warning 属于预期。

## 质量检查

演示或交付前运行：

```bash
npm run check
npm run test:api
npm run test:perf
```

当前质量门禁覆盖：

- V3.2 不渲染课程图片。
- V3.2 Profile 使用真实答题记录生成八个能力域。
- V3.2 支持课程完成 + 阶段课程题正确率大于 90% 后自动晋级，并记录晋级历史。
- 后台只读查看用户、进度、题目记录和弱项建议。
- 课程内容为 6 阶段、30 周、61 课，`missingDeepDives = 0`。
- 题库为 160 题，覆盖 61 个 lesson mapping。
- V5 题库入口复用 `question-bank-v2.js` 的数组数据结构、`answer`、`lessonMapping`、`skillClassification` 字段。
- V5 活动报名覆盖“即将开始 -> 进行中”状态切换后的已报名保持，并禁止管理员账号进入 H5 报名链路。

## 当前限制

- 不包含教官端运营工作流。
- 不包含场地装备管理。
- 不包含完整 AAR 工作流。
- 不包含成就体系。
- 后台不提供编辑、删除、重置、导出等写操作。
- 课程图片资产保留但不接入 V3.2。

## 一、任务简报系统（Mission Briefing System）

### 1.1 功能概述

每次对抗/演习/考核前，系统生成一份标准化的**任务简报**，
包含任务背景、目标设定、双方配置、规则说明、安全须知等模块。

简报在任务开始前推送到所有队员的设备（手机/平板/终端）上，
任务结束后自动归档。

## 二、失败案例库（Failure Case Library）

### 2.1 设计理念

失败案例库不是用来"羞辱"学员的——
它是**最有价值的学习资源之一**。

每一条失败案例都来源于真实的对抗/演习记录，
经过脱敏处理后用于教学分析。
学员可以从别人的错误中学习，**不需要自己亲自犯每个错误**。

### 2.2 案例格式

```json
{
  "caseId": "F-001",
  "title": "Fatal Funnel 停留阵亡",
  "stage": "W8-CQB阶段",
  "date": "2024-XX-XX",
  "location": "C号楼2层走廊",
  "participants": 6,
  "situation": 
    "小队执行房间清除任务。队员A破门进入后，
    在门口位置停顿了约1.5秒观察室内情况，
    此时室内隐蔽的敌方射手从门框右侧射击，A被击中阵亡。",
  "rootCause":
    [
      "概念理解不深：虽然学过Fatal Funnel理论但实际操作中
       仍习惯性在门口停顿观察",
      "训练不足：平时练习时没有足够压力下进行进门训练",
      "肌肉记忆未形成：紧急情况下 reverted to old habit"
    ],
  "consequence": "A阵亡→任务暂停→重新部署损失时间5min",
  "lessonLearned":
    [
      "进门后的唯一正确动作是：不停留直接移动到最近角落",
      "可以通过'进门即冲刺到角'的专项训练来强化这一本能",
      "建议增加'限时进门'训练：要求2秒内从门到角"
    ],
  "preventionMethod":
    [
      "W8课程中增加'压力进门'环节",
      "每次CQB训练前复习Fatal Funnel概念",
      "建立'进门停顿=做20个俯卧撑'的惩罚机制（温和版）"
    ],
  "relatedCourse": ["W8-L1", "W8-L2", "W17-L1", "W17-L2"],
  "tags": ["CQB", "Fatal Funnel", "房间清除", "致命错误"],
  "severity": "高"

```

### 2.4 使用方式

1. **课前引入**：讲相关课程时引用对应案例作为"反面教材"
2. **考后分析**：如果有人在考核/对抗中犯了类似错误，
   直接关联到对应案例："你犯了和F-01一样的错"
3. **定期回顾**：每月随机抽取3-5个案例全班讨论
4. **个人错题本关联**：将个人的失误与案例库条目关联，
   形成"我的成长轨迹图"

### 2.5 技术实现要点

| 模块         | 说明                                    |
|--------------|---------------------------------------|
| 案例录入界面   | 教练后台填写/批量导入JSON            |
| 标签系统       | 多维度标签（阶段/类型/严重度/课程关联） |
| 搜索筛选       | 按阶段/类型/标签/关键词搜索         |
| 统计面板       | 各类错误发生频率趋势（预防性教学依据） |
| 关联推荐       | 学员犯错后自动推荐相关案例           |
| 匿敏处理       | 案例中不含真实姓名（用代号如"队员A"）|

## 三、等级与晋升体系（Rank & Promotion System）

### 3.1 设计原则

等级体系不是"淘汰机制"而是**成长路径**。
每个等级都有明确的能力标准和可预期的晋升路径。
晋升基于**综合评估**而非单一维度（如仅看射击成绩或仅看对抗胜负）。

### 3.2 六级体系定义

#### ⭐ L0：新兵（Recruit）—— 入门~W6结束

**定位**：完成基础训练，掌握基本技能的初学者

**核心能力标准**：
- [ ] 能独立完成三姿据枪且姿势规范
- [ ] 理解并能使用10种以上基础手语
- [ ] 了解四种基础队形的名称和用途
- [ ] 能在教练口令下完成基本动作
- [ ] 通过安全知识测试（100分及格）

**晋升条件**：完成W1-W6全部课程 + 阶段一考核≥70分

**特权**：可参与基础对抗（有教练监督）/ 可借用标准装备

---

#### ⭐⭐ L1：正式队员（Operative Member）—— W7-W10结束

**定位**：具备单兵作战能力，能在班组中承担基础角色的合格战士

**核心能力标准**：
- [ ] 三姿切换流畅（任意两姿态间≤2秒）
- [ ] 能独立完成Slicing the Pie标准流程
- [ ] 理解并能在监督下执行房间清除
- [ ] 掌握Rush和匍匍的基本技术
- [ ] 能正确使用Prowords和SALUTE格式
- [ ] 射击精度达到：立姿80%+/跪姿90%+/卧姿95%+

**晋升条件**：完成W7-W10 + 阶段二考核≥75分 + 无重大安全违规

**特权**：可参与正式对抗赛/ 可申请特定装备偏好/ 有资格担任临时小组负责人

---

#### ⭐⭐⭐ L2：骨干队员（Core Member）—— W11-W14结束

**定位**：熟练的班组协作者，三三制/交替掩护等班组技能过关

**核心能力标准**：
- [ ] 能在二人组中流畅执行Bounding Overwatch（无需提醒）
- [ ] 掌握三三制阵型建立、保持和基本变换
- [ ] 在对抗中能做出正确的情境判断（不是只会跟着跑）
- [ ] Prowords/SALUTE/手语使用准确率≥90%
- [ ] 能独立带领2-3人小队完成简单任务

**晋升条件**：完成W11-W14 + 阶段三考核≥78分 + 至少1次对抗中担任临时指挥

**特权**：可带队参加外部对抗/ 有资格担任副队长/ 可优先选择训练方向（狙击/突击/防御专精）

---

#### ⭐⭐⭐⭐ L3：战术专家（Tactical Expert）—— W15-W18结束

**定位**：精通攻防SOP和特殊任务的战术多面手

**核心能力标准**：
- [ ] 能独立规划并执行完整的进攻五步SOP
- [ ] 能独立构建并守住一个防御阵地≥10分钟
- [ ] 建筑清搜流程熟练（单人或带组）
- [ ] 三种特殊任务模式（夺旗/护送/脱离接触）都能胜任
- [ ] 在压力下仍能保持60%以上的决策正确率

**晋升条件**：完成W15-W18 + 阶段四考核≥82分 + L3指挥评估通过 + 同级互评排名前50%

**特权**：可任正式班长/ 有权制定训练计划建议/ 优先获得新装备/ 外部对抗核心成员

---

#### ⭐⭐⭐⭐⭐ L4：指挥官（Officer）—— W21-W24结束

**定位**：能独立指挥一支小队的领导者，具备战略思维

**核心能力标准**：
- [ ] L3独立指挥认证通过
- [ ] 能制定完整的任务计划和应急预案
- [ ] 在Mega Game等高压环境下保持决策质量
- [ ] 具备良好的AAR组织和引导能力
- [ ] 能指导和帮助低级别队员提升

**晋升条件**：完成W21-W24 + 阶段五考核≥85分 + 
       至少2次担任正式指挥且评估"合格"以上 +
       教练组推荐

**特权**：协助教练管理班级事务/ 可代表队伍对外交流/ 
       有权提名下一届干部人选

---

#### ⭐⭐⭐⭐⭐⭐ L5：教官候选人（Instructor Candidate）—— W25-W28结束

**定位**：不仅自己强还能教别人强的准教官

**核心能力标准**：
- [ ] 全部六阶段课程内容精通
- [ ] 能识别和纠正他人的常见错误
- [ ] 具备课程设计和教案编写基础能力
- [] 在指导他人时有耐心和沟通技巧
- [ ] 个人技术能力保持在L4水准以上

**晋升条件**：完成W25-W28 + 结业考核≥90分 + 
       教学能力评估通过（试讲评分）+
       班级民主投票支持率≥70%

**特权**：可在教练监督下执教新班/ 参与教材编写/
       享有更高的装备和场地优先使用权

---

#### 🏆 L6：荣誉毕业（Graduate with Honor）—— W30结束

**定位**：30周完整训练的合格毕业生，具备独立作战和初步指挥能力

**核心能力标准**：
- [ ] 结业考试通过（≥75分）
- [ ] 全部30周课程无缺勤（特殊情况除外需补课）
- [ ] 无任何重大安全违规记录
- [ ] AAR参与率≥90%且贡献积极
- [ ] 获得至少3名以上队友/教练的推荐认可

**结业待遇**：
- 正式结业证书（含等级评定）
- 正式队服/徽章/标识
- 进入校友网络（可受邀参加后续进阶活动）
- 优秀毕业生可获得推荐信/评价

**持续发展路径**（毕业后）：
- L6→ 助教 → 教练 → 高级教练 → 总监
- L6→ 专业分队（狙击手/突击手/通讯专家）
- L6→ 对外比赛代表队

## 四、成就系统（Achievement System）

### 4.1 设计理念

成就 = 游戏化的里程碑记录。
它不是为了制造虚荣感——而是**让进步可见化、让努力有仪式感**。
每个成就都应该是**真正有意义的行为**而非" Participation Trophy"（ participation trophy）。

### 5.3 数据驱动的教学应用

**数据如何改善教学**：

1. **预测性干预**
   - 如果数据显示"全班切派脚步错误率突然上升"→ 下周重点加练切派
   - 如果某阶段考核普遍偏低 → 回顾是否教学方法需要调整

2. **个性化处方**
   - 为每个人生成"能力诊断报告"+针对性训练建议
   - 识别每个人的"天花板"（长期难以突破的瓶颈）

3. **公平性保障**
   - 对抗中的客观数据（裁判系统日志）减少争议
   - 晋升评估的多维度降低单一因素影响

4. **课程迭代依据**
   - 哪些课程内容学员普遍反映难懂 → 需要改写
   - 哪些技能在实际对抗中几乎用不到 → 考虑调整优先级
